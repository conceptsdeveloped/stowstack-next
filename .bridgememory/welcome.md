# Welcome to BridgeMemory

This hub lives at `/Users/blake/Documents/stowstack-next`.

Each memory is a Markdown file inside `.bridgememory/`. Connect them by
wrapping another memory's title in double brackets:

```
[[Another Memory]]
```

You can also alias a link with `[[Target|Display Text]]`.

## Try it

Hit "+ Memory" in the sidebar to create your first connected note.
